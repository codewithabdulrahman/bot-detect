== Description ==

Detect Bot and display content accordingly with these shortcodes:
1- [google_bot_hide text=""][/google_bot_hide]
2- [google_bot_show text=""][/google_bot_show]

== Installation ==

Install Bot Detect either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
